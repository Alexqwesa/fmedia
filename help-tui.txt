fmedia Terminal UI commands:

Key    Command
--------------
Space  Play/Pause
s      Stop
n      Next track
p      Previous track
q      Quit
h      Show help
T      Save this track from radio

Right  Seek forward
Left   Seek backward
Supported key modifiers:
  +Alt   medium step
  +Ctrl  large step

Up     Volume up
Down   Volume down
m      Mute

i      Show tags
d      Remove the track from playlist
D      Delete file from disk
